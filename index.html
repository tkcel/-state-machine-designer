ステートマシン図:

状態一覧:

待機中

自動販売機が待機状態にあり、お金の投入を待っています。

type Waiting = { totalMoney: number; }

初期値:

{ totalMoney: 0 }

お金投入済み

お金が投入され、商品選択を待っている状態です。

type MoneyInserted = { totalMoney: number; }

初期値:

{ totalMoney: 100 }

商品選択済み

商品が選択され、在庫確認中の状態です。

type ProductSelected = { totalMoney: number; selectedProduct: string; }

初期値:

{ totalMoney: 100, selectedProduct: 'コーラ' }

商品出荷中

選択された商品を出荷している状態です。

type DispensingProduct = { totalMoney: number; selectedProduct: string; }

初期値:

{ totalMoney: 100, selectedProduct: 'コーラ' }
